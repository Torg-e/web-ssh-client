### Roadmap

✅Central host management with favorites, color tags, and encrypted credential storage

✅ Web terminal with xterm.js, customizable themes, and session reconnect

✅ SFTP file browser for uploading and downloading files directly in the browser

✅ Session recording with complete terminal output and command history

✅ AI assistant supporting OpenAI, Google Gemini, and Ollama (local or cloud)

✅ Command database with preinstalled Linux commands and custom entries

✅ SSH key generation and one-click deployment to hosts

✅ Customizable terminal themes with 7 presets and background images

✅ Session export to Markdown documentations using AI


### Currently working on changes/bug fixes

- The command database will be fully customizable
- Different host card versions can be set: compact, normal, and large.
